# Παρουσίαση Βίντεο — Γραφείο Τελετών Σταυρακάκη

Κινηματογραφική, αυτο-επαναλαμβανόμενη παρουσίαση (μαύρο + λευκό + χρυσαφί)
σχεδιασμένη για την τηλεόραση Samsung του γραφείου. Υλοποιεί **ακριβώς** το
κλειδωμένο storyboard, συνολικής διάρκειας **02:50**, με fade μεταβάσεις 1 sec
και διακριτική μουσική υπόκρουση.

## Πώς παίζει

1. Ρίξτε τις φωτογραφίες και τη μουσική στον φάκελο [`assets/`](assets/README.md)
   (δείτε τα ακριβή ονόματα αρχείων εκεί).
2. Ανοίξτε το `index.html`:
   - **Στον υπολογιστή:** διπλό κλικ στο `index.html` → πατήστε **Enter** για πλήρη οθόνη.
   - **Στην τηλεόραση Samsung:** ανοίξτε το URL στον browser της TV, πατήστε
     το κουμπί ▶ «Έναρξη παρουσίασης» (ξεκινά εικόνα **και** ήχος).
   - Σε λειτουργία kiosk χωρίς αλληλεπίδραση, ξεκινά μόνο του (χωρίς ήχο) μετά
     από ~1 δευτερόλεπτο.
3. Παίζει σε **ατέρμονο loop** χωρίς απότομο τέλος.

Το κουμπί 🔊 (κάτω δεξιά) ανοίγει/κλείνει τον ήχο.

## Αν χρειαστείτε αρχείο MP4 (4K)

Η παρουσίαση είναι web ώστε να ενημερώνεται εύκολα. Αν χρειαστείτε «σκληρό»
αρχείο `.mp4`:

1. Ανοίξτε σε πλήρη οθόνη (1920×1080 ή 3840×2160 ανάλυση οθόνης).
2. Κάντε screen-record έναν πλήρη κύκλο (02:50) — π.χ. με OBS Studio
   (δωρεάν): ρυθμίστε Canvas 3840×2160, 30 fps, εξαγωγή MP4 (H.264).

Έτσι παίρνετε MP4 4K / 30fps που πληροί τις τεχνικές προδιαγραφές.
